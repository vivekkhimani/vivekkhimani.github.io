Retrocade: Arcade Gaming on Cardboard

  • Position: Python Developer
  • Type: Hackathon Project
  • Duration: March 2019
  • Location: Drexel University
  • Award: Best IoT Hack
  • Visit: https://devpost.com/software/retrocade

Soldered and multiplexed LEDs on the cardboard to build an arcade gaming screen. Used arduino to control the LEDs. I programmed games using Python which can be played using standard joystick inputs. Developed games like Apple Catcher, Snake, Pong, etc.
